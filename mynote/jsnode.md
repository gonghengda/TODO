# Math...熟悉用法
> Math.trunc()
    ```   
        Math.trunc(13.37);    // 13
        Math.trunc(42.84);    // 42
        Math.trunc(0.123);    //  0
        Math.trunc(-0.123);   // -0
        Math.trunc('-1.123'); // -1
        Math.trunc(NaN);      // NaN
        Math.trunc('foo');    // NaN
        Math.trunc();         // NaN
    ```

>Math.floor()
    ```   
        Math.floor( 45.95); //  45
        Math.floor( 45.05); //  45
        Math.floor(  4   ); //   4
        Math.floor(-45.05); // -46 
        Math.floor(-45.95); // -46
    ```

>Math.ceil()
    ```   
        Math.ceil(.95);    // 1
        Math.ceil(4);      // 4
        Math.ceil(7.004);  // 8
        Math.ceil(-0.95);  // -0
        Math.ceil(-4);     // -4
        Math.ceil(-7.004); // -7
    ```

>Math.round()   
    ```   
        Math.round( 20.49); //  20
        Math.round( 20.5);  //  21
        Math.round( 42  );  //  42
        Math.round(-20.5);  // -20
        Math.round(-20.51); // -21
    ```
